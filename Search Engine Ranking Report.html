3. What is a tag? What is the purpose for Keyword and Description meta tags?

A tag is a piece of text that describes the content of the page; the tag is not displayed on the page itself, but only in the source code of the page. The Meta elements are typically used to specify page descriptions, keywords, authors of documents, last modified, and other metadata.

4. Why might you include meta tags in your pages? Do meta tags make your pages more easily searchable or improve its ranking?

The tag is an important part because it can be used for search engines to index web pages based on title, description, and even keywords. In addition, the meta tags can make pages more easily searchable because meta tags can potentially help your SEO rankings.
